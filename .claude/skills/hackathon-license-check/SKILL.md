# ハッカソン ライセンス適合チェック

関西電力×VOLTMIND「空間を理解する」ハッカソンのルールに基づいて、モデルまたはデータセットのライセンス適合性を確認します。

## ハッカソンのモデルルール

- **禁止**: クローズドな商用モデルAPI（GPT-4, Claude, Gemini等）の推論用途での使用
- **許可**: コード生成・技術調査・開発相談等の補助用途でのクローズドモデルAPI使用
- **必須**: 商用利用可能なオープンモデルのみ
- **禁止**: コピーレフト（GPL等）により成果物の公開条件に制約が生じるモデル

## ハッカソンのデータルール

- **必須**: 公開データは商用利用可ライセンス（無償かつ許可なく使用できることが明示）
- **禁止**: ライセンスが不明確なデータ
- **必須**: 生成AIで合成する場合、合成データの商用利用をモデル提供元が認めていること
- **注意**: 人物データは肖像権・プライバシーへの配慮が必要

## チェック手順

1. 対象（モデル名またはデータセット名）を確認する
2. 公式ライセンスページ・HuggingFaceページを調べる
3. 以下の項目を確認する：
   - 商用利用の可否
   - コピーレフトの有無（GPL, LGPL等は要注意）
   - 学習・蒸留・合成データ生成への使用可否
4. 判定を以下の形式で出力する

## 出力形式

**判定**: 適合 / 不適合 / 要確認

| チェック項目 | 結果 | 備考 |
|---|---|---|
| ライセンス明記 | | |
| 商用利用可 | | |
| コピーレフトなし | | |
| 学習・合成利用可 | | |

**結論**: （使用可否の判断と理由）

ライセンスが不明確な場合は「使用禁止」と判定すること。
