# 成果発表会 プレゼン構成作成

2026/3/28(土) 14:00-17:00 グランフロント大阪北館3Fエナレッジでの成果発表会向け。

## 審査基準（重視度順）

| 評価軸 | 重要度 | 説明 |
|---|---|---|
| 独創性 | 最重視 | 新しい視点・課題設定・データ活用・モデル設計 |
| 技術力 | 最重視 | センサーデータ処理、GPU活用の工夫 |
| 事業性 | 重要 | 実社会・産業領域での価値 |
| プレゼン力 | 重要 | 課題→アプローチ→成果→限界→展望の流れ |
| デモ力 | 重要 | 「空間を理解している」と感じられる動作・結果 |

審査員：関西電力KOI 2名、VOLTMIND 1名、AI 1名

## 推奨プレゼン構成

1. **課題設定・動機**（2-3分）- 独創的な視点を冒頭で
2. **技術アプローチ**（4-5分）- データ、モデル、学習戦略、GPU活用の工夫
3. **成果・評価**（3-4分）- 定量評価 + デモ
4. **考察・限界**（1-2分）
5. **事業性・今後の展望**（1-2分）- 関電インフラ系への応用も意識

プロジェクトの内容を聞いて具体的なスライドアウトラインを作成する。
