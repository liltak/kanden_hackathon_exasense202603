## スライド構成

### 1. タイトル (1枚)

- **ExaSense — 工場向けエネルギー最適化ソリューション**
- サブタイトル: ドローン × 3D再構築 × 日照シミュレーション × AI分析
- チーム名・メンバー

### 2. 課題設定 — なぜ工場屋根 × 太陽光パネルか (2枚)

**評価軸: 独創性**

- **社会課題**: 2050年カーボンニュートラル目標、工場の脱炭素化が急務
- **現状の課題**:
  - 工場屋根の太陽光パネル設置は、現地調査 → 手動設計 → 見積もりに数週間〜数ヶ月
  - 屋根形状・影の影響の正確な評価が困難
  - 専門知識が必要で、中小工場では導入のハードルが高い
- **ExaSenseの提案**:
  - ドローン撮影 → 自動3D再構築 → 日照シミュレーション → AI設置提案
  - **数週間 → 数時間** に短縮
  - 関西電力管内の工場を主な対象

### 3. 技術アプローチ (3-4枚)

**評価軸: 技術力**

#### スライド3a: アーキテクチャ全体像

```
Phase 1-2 (GPU)     →    Phase 3 (CPU)      →   Phase 4 (GPU)   →   Phase 5
3D再構築+メッシュ        日照シミュレーション       VLM AI分析          WebUI
VGGT-1B-Commercial       pvlib + trimesh          Qwen2.5-VL         Gradio
```

#### スライド3b: Phase 1-2 — 3D再構築

- VGGT-1B-Commercial: 画像 → 1.7M点群 → Poisson再構成 → 20K面メッシュ
- 処理パイプライン図（Voxel → 外れ値除去 → Poisson → 平滑化 → 間引き）
- 旧パイプライン比 28.5倍の効率化

#### スライド3c: Phase 3 — 日照シミュレーション

- pvlib: 年間8,760時点の太陽位置・日射量計算
- trimesh: レイキャスティングによる面ごとの影行列
- 直達光 + 散乱光の統合 → ROI/NPV/回収期間の自動算出
- ヒートマップ可視化

#### スライド3d: Phase 4 — VLM分析

- Qwen2.5-VL + LoRAファインチューニング (Unsloth)
- シミュレーション結果の画像を入力 → 自然言語での設置提案
- 専門知識がなくても理解できるレポート生成
